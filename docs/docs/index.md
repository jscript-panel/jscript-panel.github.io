## Minimum requirements
`Windows 7` is the minimum requirement for all versions.

!!! note
	If running `Windows 7`, it should be fully updated. At the very least, `IE9`
	and the [Platform Update](https://www.microsoft.com/en-us/download/details.aspx?id=36805) need to be installed.

|Component|foobar2000|Columns UI (optional)
|:---|:---|:----|
|3.2.0+|2.0 Beta 18+|2.0.0-alpha.5+
|3.1.14+|2.0 Beta 13+|2.0.0-alpha.5+
|3.1.7+|2.0+|2.0.0-alpha.5+
|3.1.0+|2.0+|Any
|3.0.4+|1.6.6+|Any
|3.0.0 - 3.0.3|1.5+|Any

!!! note
	Most included samples require the installation of `FontAwesome` which
	you can grab [here](https://github.com/FortAwesome/Font-Awesome/blob/4.x/fonts/fontawesome-webfont.ttf?raw=true).

## Extra requirements
- The [IJSimage ApplyEffect](interfaces/IJSImage/#applyeffecteffect) method requires `Windows 10` or later.
- Displaying coloured emoji requires `Windows 10` or later.

## Samples
Since the learning curve for writing your own scripts is quite steep, there
are many included samples that don't require any knowledge at all. You
can simply pick the one you want to use from the `Samples` button in
the [Configuration Window](configuration-window/).

The [Gallery](../gallery/) has recently been updated with more detailed
setup notes / usage instructions.
